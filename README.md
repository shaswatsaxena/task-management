## Description

Task Management API.

## Installation

```bash
$ npm install
```

- Requires PostgreSQL instance running
- Provide values to all sample.env variables
- Rename sample.env to .env

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

Create a database named "e2e_test" to be used for testing.

```bash
# end to end tests
$ npm run test
```
