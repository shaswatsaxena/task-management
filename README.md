## Description

Task Management API built for [HackerEarth StackHack 1.0](https://www.hackerearth.com/challenges/hackathon/stackhack-v1/).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
